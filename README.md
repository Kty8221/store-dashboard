# Store Dashboard

대학교 편의점 운영자를 위한 관리자용 대시보드 웹 프로젝트입니다.  
로그인 페이지와 대시보드 화면을 중심으로 구성되어 있으며, 재고 현황, 수요 예측, 발주 추천 등의 정보를 한눈에 확인할 수 있도록 제작하고 있습니다.

## 프로젝트 소개

이 프로젝트는 대학 캠퍼스 편의점 운영 데이터를 보다 직관적으로 관리할 수 있도록 돕는 관리자용 웹 서비스입니다.

현재 프론트엔드 중심으로 구현되어 있으며, 다음과 같은 흐름으로 구성되어 있습니다.

- 로그인 페이지
- 운영자 대시보드
- 메뉴바를 통한 주요 기능 구분
- 로그아웃 기능
- 재고 현황 / 수요 예측 / 발주 추천 정보 표시

## 주요 기능

### 1. 로그인 기능
- 아이디와 비밀번호 입력을 통해 대시보드로 이동
- 로그인 후 관리자 화면 접근 가능

### 2. 데이터 시각화 (업데이트)
- Chart.js를 도입하여 재고 현황 및 AI 수요 예측 데이터를 직관적인 바 차트로 표시
- 적정 재고선 대비 현재고 상태를 시각적으로 확인 가능

### 3. 재고 현황 확인
- 상품별 현재 재고 확인
- 부족(Danger) / 주의(Warn) / 정상(Normal) 상태별 배지 표시

### 4. 수요 예측 정보 (AI Insight)
- 요일별 실제 수요와 AI 예측 데이터 비교 차트 제공
- 특정 이벤트(대학 축제 등)에 따른 수요 변화 예측 정보 표시

### 5. 발주 추천 및 인사이트
- 재고 부족 및 수요 예측을 기반으로 발주가 필요한 상품 자동 제안
- 실시간 수요 급증 항목에 대한 알림(Alert) 기능

### 6. 로그아웃 기능
- 메뉴바 하단 로그아웃 버튼을 통해 로그인 화면으로 복귀

## 화면 구성

- Login Page
  - 관리자 로그인 화면

- Dashboard Page
  - 메뉴바 (사이드바 최적화)
  - 재고 현황 패널 (차트 포함)
  - 수요 예측 패널 (차트 포함)
  - 발주 추천 패널
  - 인사이트 패널
